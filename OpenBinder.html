OpenBinder

Towards the end of my time at PalmSource, when it was clear that almost all of the software we had been working on was going to be thrown away, I spent some time trying to get as much of it open-sourced as possible. This eventually did get released, as the OpenBinder project.

Unfortunately, the server hosting this project was being managed by PalmSource, and at some point that machine disappeared and doesn't seem likely to ever return. Because occassionally I get e-mails asking about the code, and there was a lot of work put into it, I have decided to host my most recent copy here.

openbinder-12-28-2005.tar.gz (3.5MB) is a complete snapshot of the OpenBinder distribution.

You can also browse the documentation from that snapshot online.

Enjoy!
